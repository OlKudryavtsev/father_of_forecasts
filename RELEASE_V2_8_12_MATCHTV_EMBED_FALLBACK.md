# v2.8.12 — Match TV embed fallback

- Убран iframe-встроенный просмотр Match TV: сайт запрещает встраивание, из-за чего в PWA появлялись черные блоки, а на desktop — ошибка подключения.
- В секции `Видео` теперь показывается аккуратная карточка выбранного ролика без черного прямоугольника.
- Для просмотра используется кнопка `Смотреть на Match TV`, которая открывает официальную страницу источника.
- Версия PWA обновлена до `2.8.12`.
