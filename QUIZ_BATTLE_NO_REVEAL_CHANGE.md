# Изменение логики раскрытия ответа в `/quiz_battle`

Обновлена логика завершения каждого вопроса серии группового квиза.

## Было

После истечения 60 секунд бот всегда показывал правильный ответ.

## Стало

Бот показывает правильный ответ **только если есть хотя бы один правильный ответ**.

Если:

- никто не ответил;
- участники ответили, но все ошиблись;

то правильный ответ не раскрывается. Бот отправляет фановый текст о том, что правильных ответов не было.

## Где изменено

```text
app/services/quiz_battle.py
```

Функция:

```python
build_quiz_battle_question_result_text(...)
```

## Что проверить

В групповом чате:

```text
/quiz_battle
```

Проверить три сценария:

1. Есть правильный ответ — бот показывает правильный ответ и список ответивших верно.
2. Ответы были, но все неправильные — бот не показывает правильный ответ.
3. Никто не ответил — бот не показывает правильный ответ.
