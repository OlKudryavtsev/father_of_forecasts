# v2.8.69 — Полная сетка ЧМ-2026 без зависимости от неполных будущих матчей API

## Исправлено

- Турнирная сетка больше не зависит от того, создал ли API-Football будущие записи 1/8, 1/4, 1/2 и финала в локальной БД.
- В `Матч-центр → Турнир → Плей-офф` всегда отображаются все стадии: 1/16, 1/8, 1/4, 1/2, финал и матч за 3-е место.
- Полная сетка теперь строится как фиксированное дерево с линиями и реальными связями веток, а не как список уже импортированных матчей.
- Сервис «Когда могут встретиться?» считает встречу по фиксированной официальной сетке. В частности:
  - Германия — Франция: 1/8 финала;
  - Португалия — Бразилия: только финал.
- Виртуальные будущие карточки не пытаются открыть API деталей матча. Это устраняет запросы вида `/api/webapp/matches/undefined/details` и сообщение `[object Object]`.

## Источники данных

- API-Football продолжает быть источником подтверждённых пар, времени, площадок, результатов и деталей матчей.
- Фиксированная структура пути по стадиям встроена в приложение, чтобы корректно отрисовать сетку до появления всех будущих fixture-записей в API.

## Миграции

Не требуются.
