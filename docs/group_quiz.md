# Одиночный групповой квиз `/quiz`

`/quiz` в групповом чате запускает один вопрос с вариантами A/B/C/D.

## Обновленная логика завершения

Открытый вопрос завершается автоматически в двух случаях:

1. прошло 30 минут с момента запуска вопроса;
2. ответили все известные зарегистрированные участники проекта.

После завершения бот публикует итог вопроса.

## Почему "все участники" считаются по базе пользователей

Telegram Bot API не дает обычному боту надежно получить полный список участников группового чата. Поэтому бот считает ожидаемыми участниками тех пользователей, которые уже есть в таблице `users`, исключая системного пользователя `telegram_id=0`.

Если участник еще не запускал бота и не отвечал в квизах, бот не знает о нем и не учитывает его в автоматическом досрочном завершении.

## Что изменилось в тексте вопроса

Из сообщения вопроса убраны категория и тема, потому что они иногда выглядели как подсказка или прямо указывали на ответ.

Было:

```text
Категория: Сборные
Тема: ЧМ-2002
```

Стало:

```text
❓ Квиз от Отца прогнозов

<текст вопроса>

A) ...
B) ...
C) ...
D) ...
```

## Ручное завершение

Команда `/quiz_finish` по-прежнему доступна для ручного завершения текущего вопроса.
