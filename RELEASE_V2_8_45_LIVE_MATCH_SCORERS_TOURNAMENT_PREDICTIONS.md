# v2.8.45 — Live-матч, бомбардиры и турнирные прогнозы

## Что изменилось

### 🔴 Карточка матча в прямом эфире

- В верхней части Матч-центра вместо блока «Прогнозы на турнир» теперь появляется карточка матча, который идет прямо сейчас.
- Карточка показывает сборные, текущий счет, игровой статус/минуту и авторов голов с ассистентами.
- Данные берутся из уже используемого кэша деталей API-Football и обновляются в Mini App примерно раз в минуту, пока матч идет.
- По кнопке «Детали матча» открывается полный экран с событиями, статистикой, составами и видео.
- Когда в турнире нет live-матча, карточка не занимает место на экране.

### ⚽ Бомбардиры

- В топе бомбардиров добавлена отдельная колонка **«А»** — ассисты/голевые передачи, сразу после голов.
- Сортировка: сначала по голам по убыванию, при равенстве — по ассистам, затем по имени.
- Исправлено отображение количества сыгранных матчей: данные лидера сопоставляются с кэшированными статистиками уже сыгранных матчей. Это устраняет случаи, когда у игрока показывался устаревший счетчик или слово «Турнир» вместо числа матчей.
- Локальный резервный расчет теперь учитывает не только голы, но и ассисты из событий матчей.

### 🏆 Прогнозы на турнир

- Блок «Прогнозы на турнир» перенесен с верхней части Матч-центра во вкладку **«Прогнозы»**.
- После старта турнира блок по умолчанию свернут, поэтому не мешает ежедневным прогнозам на матчи.
- Внутри по-прежнему доступны просмотр собственных выборов, их статусов и переходы к сборным/игрокам; кнопка «Участники» открывает прогнозы участников лиги.

## База данных и переменные окружения

Миграции не нужны. Новые переменные окружения не требуются.

Для live-карточки должен продолжать работать уже существующий фоновый процесс синхронизации деталей матчей:

```text
MATCH_DETAILS_AUTOSYNC_ENABLED=true
MATCH_DETAILS_AUTOSYNC_INTERVAL_SECONDS=90
```

Значения выше являются стандартными; добавлять их в Railway нужно только если они ранее были отключены или переопределены.

## Проверка после деплоя

1. Откройте **Матч-центр** во время live-матча: должна появиться зеленая карточка «Сейчас идет».
2. Убедитесь, что текущий счет и авторы голов соответствуют экрану «Детали матча».
3. Откройте **Бомбардиры**: должны быть две колонки — «Г» и «А».
4. Откройте **Прогнозы**: блок «Прогнозы на турнир» расположен сверху и свернут по умолчанию.
