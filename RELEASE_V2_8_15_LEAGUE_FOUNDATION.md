# Release v2.8.15 — League foundation

Дата: 2026-06-17

## Что добавлено

- Добавлен фундамент мультилиг без изменения текущего UX.
- Добавлены таблицы `leagues` и `league_members`.
- Добавлен статус доступа пользователя `users.access_status`.
- Миграцией создается системная лига `Отец прогнозов`.
- Все текущие пользователи переводятся в статус `approved`.
- Все текущие пользователи добавляются в лигу `Отец прогнозов`.
- Администраторы добавляются в лигу с ролью `admin`, остальные — с ролью `member`.
- Рейтинг по умолчанию визуально остается прежним, но технически уже считается по участникам лиги `Отец прогнозов`.

## Миграция

Перед применением на production рекомендуется сделать backup БД.

```bash
psql "$DATABASE_URL" -f db/migrations/011_add_league_foundation.sql
```

## UX

Видимых изменений интерфейса на этом этапе нет. Это подготовительный слой для будущих заявок, приглашений и пользовательских лиг.
