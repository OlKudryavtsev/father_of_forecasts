# v2.8.75 — Дневная активность пользователей в аналитике

## Что изменено

- В **Администрирование → Аналитика** добавлен блок **«Активность пользователей по дням»**.
- Таблица показывает по каждому участнику число отслеживаемых действий за каждый день выбранного периода, общий итог и число активных дней.
- Участники отсортированы по убыванию общего числа действий; строка пользователя остаётся видимой при горизонтальной прокрутке таблицы.
- Для каждой строки выводится время последнего зарегистрированного действия.
- Данные используют существующий набор безопасных событий аналитики: Telegram ID, username, фактические счёты и тексты прогнозов не записываются.
- Фильтр **«Показывать действия администратора»** действует и на новую сводку.

## Деплой

Миграции и новые переменные окружения не нужны.
