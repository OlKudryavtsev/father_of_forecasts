# v3.4.4 — Квиз: iPhone-интерфейс, Банк вопросов и таймеры

## Что исправлено

- Исправлено открытие вкладки **«Банк»**: добавлено отсутствовавшее frontend-состояние `bankLoading`, из-за которого React завершал рендер с ошибкой `Can't find variable: bankLoading`.
- Убрана верхняя промо-плашка «Квиз лиги» с версией и описанием.
- Три рабочие вкладки квиза переработаны для iPhone: компактный переключатель **Игры / План / Банк**, крупные зоны нажатия, единые карточки и плотная мобильная компоновка.
- В Банке добавлены поиск по тексту, теме и тегу, а также фильтр по формату вопроса — это необходимо для удобной работы с большим набором из 305 карточек.
- В Планировании явно показано, что вопросы берутся из одобренного Банка. При отсутствии подходящих вопросов доступна кнопка **«Открыть Банк вопросов»**.
