# v3.4.7 — исправление сворачивания панелей Банка вопросов

## Проблема
В Safari/Telegram WebView на iPhone при раскрытии или сворачивании одной из панелей Банка вопросов приложение могло показывать ошибку:

`null is not an object (evaluating 'u.currentTarget.open')`

## Причина
Обработчик `onToggle` обращался к `event.currentTarget.open` внутри отложенного callback `setState`. К моменту выполнения callback React мог уже очистить ссылку на объект события.

## Исправление
Состояние `open` теперь считывается синхронно в самом обработчике события, до вызова `setState`, и затем передаётся в состояние обычным булевым значением.

Исправлены все три управляемые панели Банка:
- Загрузка готовых наборов;
- Новый вопрос в Банк;
- Импортировать вопросы из JSON.

## Деплой
Миграция не нужна. После выкладки обновите PWA до версии 3.4.7.
