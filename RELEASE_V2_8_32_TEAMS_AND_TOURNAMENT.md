# v2.8.32 — Сборные и турнир

## Что добавлено

- В Матч-центре появились режимы **«Турнир»** и **«Бомбардиры»** — без новой нижней вкладки.
- Таблицы всех групп ЧМ-2026: выбор группы, очки, мячи, зона выхода в плей-офф.
- Нажатие на сборную открывает ее профиль: положение в группе, статистика, результаты и календарь, бомбардиры.
- Добавлен рейтинг бомбардиров с фото, голами, ассистами и переходом в профиль игрока.
- В профиле игрока: команда, голы, ассисты, матчи и синхронизированные игровые показатели.
- Реализованы переходы: группа → сборная → матч; матч → сборная/бомбардир; бомбардир → игрок → сборная/матч.

## Данные и кэш

- Матчи и таблицы рассчитываются по данным приложения — это источник истины для расписания и результатов.
- Бомбардиры и профили игроков обновляются из API-Football и сохраняются в общем backend-кэше `tournament_data_cache`.
- При недоступности API приложение использует данные из уже закэшированных событий матчей.

## Миграция

```bash
psql "$DATABASE_URL" -f db/migrations/017_add_tournament_data_cache.sql
```
