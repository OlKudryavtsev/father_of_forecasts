# Admin button top and Fantasy classic squad positions

## Что изменено

### 1. Кнопка `Админ` перенесена наверх

- Вкладка `Админ` убрана из нижнего меню.
- Кнопка `Админ` теперь отображается в верхнем блоке, слева от обратного отсчета.
- Кнопка видна только пользователям с `is_admin=true`, то есть тем, чей Telegram ID входит в `ADMIN_TELEGRAM_IDS`.

### 2. Fantasy: классические ограничения по позициям

Общее ограничение заявки теперь соответствует классике:

- 2 вратаря
- 5 защитников
- 5 полузащитников
- 3 нападающих

Скамейка больше не привязана жестко к позициям `ВР запас / ЗЩ запас / ПЗ запас / ПЗ запас`.

Теперь запасные слоты универсальные:

- Запас 1
- Запас 2
- Запас 3
- Запас 4

В них можно выбрать любую позицию, если итоговая заявка остается 2/5/5/3.

### 3. Информационное сообщение для уже собранных составов

Если у пользователя уже был собран состав, который после смены схемы или старой логики не проходит по классическим позиционным лимитам, под запасными около кнопки `Сохранить команду` показывается красное сообщение:

```text
Состав не соответствует классическим ограничениям по позициям...
```

Кнопка сохранения блокируется до исправления состава.
