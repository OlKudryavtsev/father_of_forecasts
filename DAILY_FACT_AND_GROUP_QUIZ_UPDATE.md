# Изменения: daily fact и одиночный групповой `/quiz`

## Daily fact

Файл: `app/formatters/facts.py`

Из ежедневного сообщения убран блок мини-вопроса. Вместо него добавлены:

- случайная архивная карточка из `/archive`;
- ссылки на `/quiz_battle`, `/quiz`, `/fact`, `/archive`.

Случайная архивная карточка выбирается в `app/services/facts.py` из активных публичных карточек `historical_archive_cards`.

## Одиночный групповой `/quiz`

Файлы:

- `app/formatters/quiz.py`
- `app/services/quiz.py`
- `app/handlers/quiz.py`

Из текста вопроса убраны категория и тема.

Добавлено автоматическое завершение вопроса:

- через 30 минут после запуска;
- или досрочно, когда ответили все известные зарегистрированные участники.

Техническое ограничение: Telegram Bot API не позволяет надежно получить полный список участников группы, поэтому "все участники" считаются по таблице `users`, исключая системного пользователя `telegram_id=0`.

## SQL

Схема БД не менялась. SQL-миграции не требуются.
