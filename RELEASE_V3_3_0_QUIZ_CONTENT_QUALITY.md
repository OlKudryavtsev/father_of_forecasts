# v3.3.0 — Квиз, этап 4: контент и контроль качества

## Что добавлено

- Полный редактор вопроса: изменение текста, типа, вариантов, правильного ответа, баллов, алиасов, темы, подсказок, топ-10 и пояснения.
- Несколько источников у одного вопроса: название, URL и комментарий.
- Изображение по HTTPS URL с предпросмотром в редакторе и показом игрокам в PWA. Файловая загрузка намеренно не реализована: Railway-контейнер не является постоянным медиахранилищем.
- Предпросмотр вопроса до включения в квиз.
- Редактирование одобренного вопроса автоматически возвращает его в черновик. Уже созданные квизы не меняются: они используют неизменяемые снимки вопросов.
- Архивирование и восстановление вопросов.
- Журнал изменений банка: создание, импорт, редактирование, одобрение, архивирование и тестовая загрузка.
- Экспорт банка в JSON и импорт JSON как черновиков.
- Ручное рассмотрение спорных ответов после закрытия вопроса: «Зачесть» или «Отклонить» с обязательным комментарием; счёт корректируется отдельным событием и аудитом.
- Кнопка загрузки 9 одобренных вопросов по ЧМ-2026: по одному для каждого поддержанного формата.

## Миграция

Перед деплоем примените после миграций 023–025:

```sql
\i db/migrations/026_add_league_quiz_content_quality.sql
```

## Проверка

1. Откройте `Квиз → Управление квизом` от имени администратора лиги.
2. Нажмите `Загрузить вопросы ЧМ-2026`. В банке появятся 9 вопросов всех форматов.
3. Откройте один вопрос через `Предпросмотр`; проверьте тип, варианты, пояснение и ссылки.
4. Нажмите `Изменить`, измените пояснение и сохраните: вопрос должен перейти из «Одобрен» в «Черновик». Одобрите его повторно.
5. Нажмите `История`: должны быть видны события создания, редактирования и одобрения.
6. Нажмите `Экспорт JSON`, затем вставьте содержимое в раздел импорта. Импортированные вопросы должны появиться как черновики.
7. Создайте короткий текстовый квиз, ответьте намеренно близкой, но нераспознанной формулировкой. Во время раскрытия вопроса откройте `Проверить ответы`, нажмите `Зачесть`, добавьте комментарий. Таблица должна сразу пересчитать очки.

## Ограничения этой версии

- Медиа — только внешние HTTPS-изображения. Для надёжной загрузки файлов в следующем релизе потребуется настроить постоянное object storage (S3/R2/аналог).
- Ручная проверка в PWA доступна для текущего раскрытого вопроса; API уже поддерживает проверку любого закрытого вопроса по его ID.
