# Tournament tab collapsed blocks update

Обновлена вкладка Mini App «Турнир».

## Что изменено

1. `Прогноз Отца` больше не показывается сразу.
2. Из отображения `Прогноза Отца` убрана версия `v1`.
3. Порядок блоков теперь такой:
   - `Мой турнирный прогноз`
   - кнопка `Прогнозы участников`
   - кнопка `Прогноз Отца прогнозов`
   - кнопка `Подсказка по бомбардирам`
4. По умолчанию все дополнительные блоки скрыты.
5. Подсказка по бомбардирам вынесена из формы выбора бомбардира в отдельную кнопку ниже.

SQL и backend не менялись.
