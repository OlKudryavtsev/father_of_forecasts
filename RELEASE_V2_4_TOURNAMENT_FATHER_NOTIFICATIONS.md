# v2.4 Tournament lock, Father predictions and notifications

## Включено

1. Турнирные прогнозы закрыты для редактирования после старта турнира в UI.
2. Добавлен просмотр турнирных прогнозов других участников после старта турнира.
3. Добавлены прогнозы Отца прогнозов вне общего рейтинга.
4. Прогноз Отца фиксируется по матчу и после старта не меняется.
5. На первых двух матчах зафиксированы прогнозы: 1:0 и 1:1.
6. В участниках матча прогноз Отца показывается отдельно.
7. После старта матча кнопка «Прогноз Отца» скрывается с карточки матча.
8. Добавлены групповые уведомления о старте и окончании матча.
9. После окончания матча участники подсвечиваются:
   - зеленый — точный счет;
   - синий — угаданный исход;
   - красный — мимо.

## SQL

Нужно выполнить миграцию:

```bash
psql "$DATABASE_URL" -f db/migrations/006_add_father_match_predictions.sql
```

## Важно

Для групповых уведомлений должны быть включены:

```text
REMINDERS_ENABLED=true
GROUP_CHAT_ID=<id группы>
```

Цикл уведомлений использует существующий reminders loop.
