# v2.8.10 — PWA update fix, video filtering and automatic API-Football sync

## Что изменилось

### PWA/Web
- Исправлена рассинхронизация версии frontend/backend, из-за которой баннер "Доступна новая версия" мог появляться повторно после нажатия "Обновить".
- Версия приложения обновлена до `2.8.10`.
- `/api/webapp/app-version` теперь отдается с no-cache/no-store заголовками.
- Кнопка обновления PWA дополнительно очищает service worker/cache storage и перезагружает `/app` с cache-busting параметром.

### Видео в матч-центре
- В секции "Видео" теперь показываются ссылки по состоянию матча:
  - до/во время матча — только `Трансляция` (`live`);
  - после завершения матча — только `Голы и лучшие моменты` (`highlights`).
- Live-ссылка после матча больше не отображается в карточке матча.
- Иконка 🎥 в списке матчей теперь появляется только если есть видео, которое реально показывается для текущего состояния матча.

### Автоматическое обновление результатов и Fantasy
- Добавлен фоновый job автоматической синхронизации API-Football.
- Job ищет недавно начавшиеся/завершившиеся матчи, обновляет статус и результат, если API-Football вернул `FT`, `AET` или `PEN`.
- После синхронизации результатов автоматически обновляется статистика игроков Fantasy и пересчитываются fantasy-очки команд.

## Новые переменные окружения

```bash
API_FOOTBALL_AUTOSYNC_ENABLED=true
API_FOOTBALL_AUTOSYNC_INTERVAL_SECONDS=600
```

По умолчанию автосинхронизация включена и запускается раз в 10 минут.
