# Изменение `/quiz_battle`: уведомление об ответе участника

В этой версии после каждого принятого ответа в серии группового квиза бот отправляет в общий чат короткое сообщение:

```text
✅ <Имя участника> сделал свой выбор
```

## Зачем

Так участники сразу видят, кто уже включился в игру, а кто пока молчит. При этом выбранный вариант ответа не раскрывается.

## Измененный файл

```text
app/handlers/quiz.py
```

## SQL

Схема БД не изменилась. SQL-миграции применять не нужно.

## Проверка

1. В групповом чате запустить `/quiz_battle`.
2. Выбрать 3 вопроса.
3. Нажать вариант ответа.
4. Убедиться, что бот отправил в чат сообщение вида:

```text
✅ Саня сделал свой выбор
```
