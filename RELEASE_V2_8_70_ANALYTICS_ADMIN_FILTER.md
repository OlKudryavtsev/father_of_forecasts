# Отец прогнозов v2.8.70 — Аналитика без действий администратора

## Изменение

В разделе **Администрирование → Аналитика** собственные действия администратора теперь исключаются по умолчанию. Это позволяет видеть фактическое использование приложения участниками, без влияния проверок, настроек и администрирования.

Переключатель **«Показывать действия администратора»** расположен в верхней части отчёта. Включение обновляет все расчёты и историю действий одним запросом.

## Техническая реализация

- `GET /api/webapp/admin/analytics` получил параметр `include_admin` со значением по умолчанию `false`.
- При `include_admin=false` фильтруются события текущего администратора во всех агрегатах и истории.
- Изменения схемы БД, миграции и новые переменные окружения не требуются.
