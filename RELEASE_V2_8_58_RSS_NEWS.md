# v2.8.58 — Новости Отца: RSS-подборка и Telegram-дайджест

## Что добавлено

- В **Ресурсы** добавлена карточка **«😂 Новости Отца»**: необычные, фанатские, смешные и странные новости вокруг ЧМ-2026.
- Каждая карточка содержит краткий факт, комментарий Отца, дату и источник. Нажатие открывает исходный материал.
- Бот читает только RSS-ленты, не использует платный Web Search и не загружает полные страницы источников.
- OpenAI вызывается максимум один раз на сканирование: выбирает одну подходящую новость из пакета RSS-кандидатов, делает краткий пересказ и комментарий.
- Одну отобранную новость бот отправляет в каждый подключённый чат лиги. Доставка идемпотентна: повторный запуск не дублирует сообщение.
- В PostgreSQL сохраняются отобранные и отклонённые кандидаты, факт доставки и расход токенов для RSS-курации.
- Добавлен admin API: `GET /api/webapp/admin/news/usage?days=30` для контроля числа вызовов, токенов и оценочной стоимости RSS-курации.

## Переменные окружения

Минимально достаточно уже существующего `OPENAI_API_KEY`. По умолчанию включён экономный режим:

```env
NEWS_ENABLED=true
NEWS_TIMEZONE=Europe/Moscow
NEWS_SCAN_HOURS=9,15,21
NEWS_POLL_INTERVAL_SECONDS=300
NEWS_MAX_POSTS_PER_DAY=1
NEWS_MAX_AI_CALLS_PER_DAY=3
NEWS_MAX_CANDIDATES_PER_SCAN=10
NEWS_AI_ENABLED=true
```

Модель берётся в порядке: `NEWS_AI_MODEL` → `OPENAI_GAMIFICATION_MODEL` → `OPENAI_FORECAST_MODEL` → `gpt-5.4-mini`.

По умолчанию используются две тематические Google News RSS-ленты — русская и англоязычная. Их можно заменить собственным перечнем через `NEWS_RSS_FEEDS`; разделитель — `;` или перевод строки:

```env
NEWS_RSS_FEEDS=https://example.org/rss-1.xml;https://example.org/rss-2.xml
```

## Ограничения и безопасность

- Нельзя публиковать трагедии, болезни, травмы, политику, войну, дискриминацию и неподтверждённые слухи.
- OpenAI получает только заголовок, короткое RSS-описание, источник и дату; он не определяет факты и не выполняет поиск.
- Без `OPENAI_API_KEY` доступен осторожный локальный fallback: публикуется только кандидат с явными маркерами фанатского/курьёзного сюжета.
- При повторном запуске после деплоя обрабатывается только один последний пропущенный слот, а не все слоты дня подряд.
