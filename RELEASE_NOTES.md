# 🚀 Release Notes — Отец прогнозов

## 2026-05-17

### Добавлено

- **Админская рассылка Release Notes** через новую команду:

  ```text
  /admin_release group
  текст релиза
  ```

  Поддерживаемые режимы:

  ```text
  group — отправить сообщение в общий чат
  users — отправить всем зарегистрированным пользователям в личку
  both — отправить и в общий чат, и пользователям
  ```

  Пример:

  ```text
  /admin_release group
  🚀 Release Notes v0.7

  Добавлено:
  — /quiz_battle — серия квиза на 3/5/10 вопросов
  — /table_buttons — компактная таблица кнопками
  ```

- **Групповой квиз-баттл `/quiz_battle`**:
  - выбор количества вопросов: 3 / 5 / 10;
  - 60 секунд на каждый вопрос;
  - автоматический переход к следующему вопросу;
  - итоговая таблица участников после завершения серии.

- **Новые таблицы БД для серии квиза**:
  - `group_quiz_games`;
  - `group_quiz_game_questions`;
  - `group_quiz_game_answers`.

- **Документация по квиз-баттлу**:
  - `docs/quiz_battle.md`;
  - `QUIZ_BATTLE_IMPLEMENTATION.md`.

- **DB-документация и bootstrap-скрипты**:
  - `db/schema.sql`;
  - `db/README.md`;
  - `docs/database.md`;
  - `scripts/init_db.py`;
  - `scripts/check_db_schema.py`.

### Изменено

- В `/quiz_battle` изменена логика показа правильного ответа:
  - если есть хотя бы один правильный ответ — правильный ответ показывается;
  - если никто не ответил — правильный ответ не раскрывается;
  - если ответы были, но все неправильные — правильный ответ не раскрывается.

- Обновлена документация проекта:
  - расширен `README.md`;
  - добавлено описание БД, команд, архитектуры и функциональности бота.

### Технические изменения

- Проект переведен на модульную структуру:
  - `handlers/`;
  - `services/`;
  - `keyboards/`;
  - `formatters/`;
  - `constants/`;
  - `middleware/`;
  - `jobs/`.

- Удален compatibility-layer после модульного рефакторинга.
- Удалены неиспользуемые MVP-файлы, старые predictor-модули, IDE/cache-файлы и временные артефакты.
- Добавлен `.gitignore`.

## 2026-05-16

### Добавлено

- **Panini-режим `/panini`**:
  - генерация футбольной карточки игрока сборной по фотографии;
  - выбор сборной из топ-20 участников ЧМ-2026 по FIFA ranking;
  - поддержка работы в групповом чате;
  - OpenAI image generation/edit.

- **Компактная таблица кнопками `/table_buttons`**:
  - удобный формат для просмотра рейтинга на телефоне;
  - отдельные колонки для очков, точных счетов, исходов и турнирных очков.

- **Групповые уведомления**:
  - о первой регистрации участника;
  - о факте сделанного прогноза на матч;
  - о факте сделанного турнирного прогноза.

### Изменено

- Формат вывода матчей изменен на более удобный для мобильных кнопок:

  Было:

  ```text
  Группа A. Тур 1. Мексика 🇲🇽 — 🇿🇦 ЮАР
  ```

  Стало:

  ```text
  Мексика 🇲🇽 — 🇿🇦 ЮАР. Тур 1. Группа A
  ```

- Расширены команды, доступные в общем чате:
  - `/matches_all`;
  - `/forecast`;
  - `/match`;
  - `/predictions`;
  - `/table`;
  - `/table_buttons`;
  - `/tournament_predictions`;
  - `/panini`.

- Для `/forecast` добавлен выбор матчей ближайших трех игровых дней.

### Исправлено

- Исправлены проблемы с отображением русских/английских названий сборных.
- Исправлены флаги для команд в списках матчей.
- Исправлена ошибка форматирования уведомлений о прогнозах из-за отсутствующего поля `round_number`.


## Обновление: ежедневный факт и одиночный групповой квиз

### Изменено

- В ежедневной рубрике убран мини-вопрос.
- В ежедневную рубрику добавлена случайная карточка из архива `/archive`.
- В ежедневную рубрику добавлены понятные CTA на `/quiz_battle`, `/quiz`, `/fact` и `/archive`.
- В одиночном групповом `/quiz` убраны категория и тема из текста вопроса, чтобы они не подсказывали ответ.
- Одиночный групповой `/quiz` теперь автоматически завершается через 30 минут.
- Одиночный групповой `/quiz` завершается досрочно, если ответили все известные зарегистрированные участники.
