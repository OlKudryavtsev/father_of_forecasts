# v2.8.20 — Public user league scope fix

Дата: 2026-06-17

## Что изменено

- Новые пользователи после одобрения больше не добавляются автоматически в системную лигу «Отец прогнозов».
- Если пользователь пришел по invite-code, после одобрения он добавляется только в приглашенную лигу.
- Если пользователь пришел без invite-code, после одобрения он получает доступ к боту, но остается без активной лиги до создания своей лиги или вступления по коду.
- Уведомления о заявках и одобрении новых пользователей отправляются только администраторам в личные сообщения с ботом.
- Убрана отправка уведомлений о регистрации/одобрении новых пользователей в групповой чат.
- Групповые уведомления о сделанном прогнозе и турнирном прогнозе теперь отправляются только по участникам системной лиги «Отец прогнозов».

## Миграции

Миграции БД не требуются.
