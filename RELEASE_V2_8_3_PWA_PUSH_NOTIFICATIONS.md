# Release v2.8.3 — PWA Push Notifications

Изменения:

- Добавлены реальные типы PWA push-уведомлений:
  - напоминания о прогнозах;
  - активность участников;
  - старт матча;
  - итоги матча;
  - ежедневная сводка / факт дня.
- Удалены из UI настройки уведомлений, для которых не было фактической отправки: Release Notes, квизы, Fantasy.
- Push-отправка теперь учитывает:
  - глобальные переключатели администратора;
  - персональные настройки пользователя в профиле;
  - активность PWA-подписки пользователя.
- Админский раздел “Push-уведомления” сохранен: можно отправить тестовое push-уведомление текущему администратору.
- Обновлены frontend-исходники и собранная директория `app/miniapp_static`.
