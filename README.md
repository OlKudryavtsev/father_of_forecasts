# 🏆 Отец прогнозов — Telegram-бот для турнира прогнозов ЧМ-2026

**Отец прогнозов** — это Telegram-бот для дружеского турнира прогнозов на матчи Чемпионата мира 2026.  
Бот помогает участникам делать прогнозы на матчи и итоги турнира, считает очки, ведет таблицу, показывает статистику, присылает факты о чемпионатах мира, проводит квизы, генерирует ИИ-прогнозы и даже делает футбольные карточки в стиле Panini по фотографии.

Проект вырос из таблиц Google Sheets для прошлых турниров и превратился в полноценного Telegram-бота с базой данных, интеграцией API-Football и OpenAI.

---

## 📌 Основная идея

Участники заранее прогнозируют:

- счет каждого матча;
- победителя/проход в плей-офф;
- чемпиона турнира;
- финалиста;
- третье место;
- лучшего бомбардира.

Бот автоматически:

- принимает прогнозы;
- закрывает прием после стартового свистка;
- скрывает прогнозы до начала матча;
- считает очки после внесения результата;
- строит таблицу участников;
- показывает, кто забыл сделать прогноз;
- ведет развлекательные рубрики до старта турнира.

---

## ⚽ Основные возможности

### 1. Прогнозы на матчи

Команда:

```text
/predict
```

Бот показывает ближайший игровой день и предлагает выбрать матч кнопками.

Пример:

```text
#1. Мексика 🇲🇽 — 🇿🇦 ЮАР. Тур 1. Группа A
Старт: 11.06.2026 22:00
```

После выбора матча участник выбирает счет кнопками:

```text
0:0 | 1:0 | 0:1
1:1 | 2:0 | 0:2
2:1 | 1:2 | 2:2
```

Также доступен ручной ввод счета.

Прогноз можно менять сколько угодно раз **до начала матча**.

---

### 2. Скрытие прогнозов до старта матча

До стартового свистка участники видят только факт, что прогноз сделан:

```text
Саня: ✅ прогноз сделан
Пикан: ❌ прогноза нет
```

После начала матча прогнозы раскрываются:

```text
Саня: 2:1
Пикан: 1:1
Антонио: 0:1
```

---

### 3. Плей-офф и прогноз на проход

В матчах на вылет можно дополнительно поставить, кто пройдет дальше.

Система очков:

```text
угадал счет/исход — стандартные очки
угадал проход — +1
не угадал проход — -1
не ставил на проход — 0
```

Пример:

```text
Прогноз: Аргентина — Франция 2:2
Проход: Аргентина
```

Команда:

```text

```

Бот пошагово спрашивает:

```text
2. Финалист
3. 3 место

```

Прогноз на турнир можно менять до старта турнира.

До старта турнира детали скрыты.

## 👥 Групповой квиз

В общем чате команда:

```text
/quiz
```

запускает общий вопрос для всех участников.

Особенность группового квиза:

- ответы принимаются скрыто;
- каждый участник может ответить только один раз;
- правильный ответ раскрывается после завершения;
- бот показывает, кто ответил правильно, а кто промахнулся.

Одиночный вопрос завершается автоматически через 30 минут или раньше, если ответили все зарегистрированные участники группы.

Завершить вопрос вручную:

```text
/quiz_finish
```


В `/quiz_battle` после каждого принятого ответа бот публикует в чат короткое сообщение:

```text
✅ Саня сделал свой выбор
```

Так видно, кто уже играет в текущем вопросе, но выбранный вариант ответа остается скрытым.

Рейтинг группового квиза:

```text
/quiz_table
```

Пример результата:

```text
🏁 Квиз завершен

Правильный ответ: B) Мирослав Клозе

✅ Верно ответили:
Пикан, Саня

❌ Мимо:
Вадик, Антонио

🔥 Отец прогнозов:
Кто ответил верно — красавчики. Кто мимо — добро пожаловать в зону прогнозов 1:1.
```

### 🏆 Квиз-баттл с таймером

Команда для группового чата:

```text
/quiz_battle
```

Логика:

1. Бот предлагает выбрать количество вопросов: 3, 5 или 10.
2. Каждый вопрос показывается в общем чате.
3. На ответ дается 60 секунд.
4. Если был хотя бы один правильный ответ, бот показывает правильный ответ. Если никто не отвечал или все ошиблись, правильный ответ не раскрывается.
5. Следующий вопрос появляется автоматически, даже если ответили не все.
5. После последнего вопроса бот показывает итоговую таблицу.

Пример:

```text
❓ Вопрос 1/5

Кто выиграл ЧМ-2002?

A) Германия
B) Бразилия
C) Франция
D) Италия

⏳ На ответ: 60 секунд
```

Финал:

```text
🏁 Квиз-баттл завершен

Вопросов: 5

Итоги:
1. Пикан — 4/5 (38 сек.)
2. Саня — 3/5 (45 сек.)
```

## 🧮 Система начисления очков

### Матчевые прогнозы

Типовая логика:

```text
точный счет — 3 очка
угаданный исход — 1 очко
не угадал — 0 очков
```

Для плей-офф дополнительно:

```text
угадал проход — +1
не угадал проход — -1
не ставил на проход — 0
```

### Турнирные прогнозы

Очки начисляются за:

```text
чемпиона
финалиста
третье место
лучшего бомбардира
```

Конкретная логика находится в модуле:

```text
app/scoring.py
```

- Для матчей плей-офф доступны счёт и необязательный выбор команды на проход; очки и рейтинг пересчитываются сразу после сохранения.
- Ручное выставление результата удалено из интерфейса Mini App. Основной путь обновления результатов — API-Football.

## v2.8.73

- Исправлено начисление за проход в плей-офф: при точном счёте и неверном проходе теперь получается `3 − 1 = 2` очка.
- Автосинхронизация определяет победителя и по результату серии пенальти / дополнительного времени, даже когда API временно не заполнил флаг победителя команды.
- В шапке Mini App и на карточках матчей стадия `Round of 32 / 1/16` отображается как `1/16 финала`, а не `1/8 финала`.
