# Release v2.8.4 — видео в матч-центре

## Добавлено

- Новый ручной раздел видео для матчей.
- Таблица `match_videos` для официальных ссылок на трансляции, обзоры, хайлайты и записи.
- Backend API для администратора:
  - `GET /api/webapp/admin/matches/{match_id}/videos`
  - `POST /api/webapp/admin/matches/{match_id}/videos`
  - `PUT /api/webapp/admin/match-videos/{video_id}`
  - `DELETE /api/webapp/admin/match-videos/{video_id}`
- В админке для выбранного матча добавлен блок `Видео матча`:
  - тип видео;
  - название;
  - ссылка;
  - порядок отображения;
  - признак показа/скрытия.
- В матч-центре добавлен компактный блок `Видео`, который показывается только если у матча есть активные ссылки.
- Для live-трансляций добавлен визуальный бейдж `live`.

## Миграция БД

Для существующей production-БД нужно выполнить:

```sql
\i db/migrations/008_add_match_videos.sql
```

Или применить SQL из файла миграции вручную через Railway/PostgreSQL console.

## Примечание

Этап 1 использует только ручное добавление официальных ссылок. Автоматический поиск обзоров Match TV не включен в эту версию.
