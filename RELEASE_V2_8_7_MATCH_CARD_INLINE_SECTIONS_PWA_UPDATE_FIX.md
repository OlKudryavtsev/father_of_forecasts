# Release v2.8.7 — Match card inline sections and PWA update fix

## Что изменено

### PWA update flow
- Исправлена механика кнопки «Обновить» в баннере новой версии.
- При обновлении приложение:
  - переводит кнопку в состояние «Обновляю…»;
  - обновляет и снимает регистрацию старого service worker;
  - очищает Cache Storage;
  - открывает `/app` с cache-busting параметром.
- Версия frontend/service worker/backend обновлена до `2.8.7`.

### Видео в матч-центре
- Секция «Видео» теперь свернута по умолчанию.
- Секция занимает всю ширину карточки матча.
- Внутри видео больше не показывается HTML-разметка и длинное название матча.
- Вместо этого показывается понятный тип видео:
  - «Трансляция»;
  - «Голы и лучшие моменты»;
  - «Обзор матча»;
  - «Полная запись»;
  - «Гол»;
  - «Момент».
- Источник отображается аккуратно как `Match TV`.

### Прогнозы участников
- Кнопка «Участники» убрана из ряда действий матча.
- Добавлена секция «Прогнозы участников» внутри карточки матча.
- Секция свернута по умолчанию и раскрывается ниже в карточке, без модального окна.
- При раскрытии подгружаются прогнозы участников.

## Миграции
Новые миграции БД не требуются.
