# Fantasy player picker dropdown update

## Что изменено

В Fantasy выбор игрока переделан на два выпадающих списка:

1. `Сборная`
2. `Игрок`

Теперь игрока не нужно искать вручную через текстовый поиск.

## Логика

- Список сборных показывает только те команды, где есть игроки нужной позиции для выбранного слота.
- После выбора сборной становится доступен список игроков.
- Игроки, которых нельзя выбрать из-за лимитов, отображаются в списке как недоступные:
  - `уже выбран`
  - `лимит сборной`
  - `лимит категории`
- После выбора игрока он сразу ставится в выбранный слот.
- Блок текущего игрока остался: можно сделать капитаном или убрать игрока.
