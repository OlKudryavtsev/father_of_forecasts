# v2.8.17 — Интерфейс лиг

Релиз добавляет третий этап мультилиг без изменения модели прогнозов: прогноз на матч по-прежнему один на пользователя, а рейтинги и списки участников фильтруются по выбранной лиге.

## Что добавлено

- В PWA добавлен раздел **Лиги**.
- Пользователь видит список своих лиг и активную лигу.
- Можно создать частную лигу.
- Можно вступить в лигу по invite-code.
- Для лиг с правами управления отображается invite-code / приглашение для копирования.
- В **Рейтинге** добавлен переключатель активной лиги.
- В **Матч-центре** добавлен переключатель активной лиги.
- В секции **Прогнозы участников** внутри карточки матча показываются только участники выбранной лиги.
- Модальное окно **Прогнозы участников на турнир** также фильтруется по активной лиге.

## Что не менялось

- Прогнозы не дублируются по лигам.
- Старые прогнозы, очки, fantasy и турнирные прогнозы сохраняются.
- Пользователь может состоять в нескольких лигах, но прогноз на матч остается один.
- Системная лига **Отец прогнозов** остается лигой по умолчанию.

## Миграции

Новых миграций БД нет. Используются таблицы `leagues` и `league_members`, добавленные в v2.8.15.
