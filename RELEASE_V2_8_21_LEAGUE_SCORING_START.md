# v2.8.21 — League scoring start

## Что изменено

- Добавлено поле `leagues.scoring_start_at` — точка старта подсчета очков для лиги.
- Для новых частных лиг `scoring_start_at` устанавливается в момент создания лиги.
- Для системной лиги «Отец прогнозов» старт подсчета установлен на начало турнира, чтобы текущий рейтинг не изменился.
- Для участников системной лиги дата вступления также нормализуется к старту турнира, чтобы старые прогнозы продолжили отображаться в матч-центре.
- Рейтинг выбранной лиги теперь считает матчевые очки только по матчам, начавшимся после `scoring_start_at` лиги.
- Турнирный прогноз учитывается в новых лигах, как и просили.
- В матч-центре раздел «Прогнозы участников» показывает только участников, которые уже состояли в выбранной лиге на момент начала матча.
- В разделе «Лиги» теперь отображается, с какого момента считается рейтинг.
- PWA-версия обновлена до `2.8.21`.
