# Release v2.8.8 — Match video UX and push notifications

## Что добавлено

- Push-уведомление при появлении нового активного видео матча Match TV:
  - хайлайты;
  - обзор;
  - полная запись.
- Новая настройка уведомлений: **Видео матчей**.
- Бейдж **Видео доступно** в карточке матча.
- Иконка 🎥 в верхней строке карточки матча, если к матчу привязано видео.

## Технические изменения

- Добавлено поле `match_videos.notification_sent_at`, чтобы не отправлять повторные push по одному и тому же видео.
- Автосинк Match TV теперь возвращает счетчик `push_notifications_sent`.
- Версия PWA обновлена до `2.8.8`.

## Миграция

```bash
psql "$DATABASE_URL" -f db/migrations/010_match_video_push_notifications.sql
```
