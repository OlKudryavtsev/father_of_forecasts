# v2.8.56 — AI-разбор открывшихся прогнозов

## Что добавлено

После стартового свистка бот отправляет в Telegram-чат каждой настроенной лиги один общий пост на матч или на группу матчей с одинаковым временем начала.

В сообщении:

- раскрытые прогнозы всех участников лиги;
- распределение по исходам: победа хозяев, ничья, победа гостей;
- единое мнение лиги и уникальные прогнозы;
- текущая таблица перед стартом;
- форма каждого участника по трём последним завершённым прогнозам;
- ближайшие дуэли в таблице, когда участники идут рядом и выбрали разные сценарии;
- короткий AI-разбор «Отца прогнозов» в настроенном для лиги тоне. По умолчанию — «Без пощады».

## Поведение при одновременных матчах

Матчи с одинаковым временем начала объединяются в одно сообщение. Поэтому в групповом чате не будет серии почти одинаковых постов при параллельном игровом слоте.

## Надёжность после деплоя

Для разбора работает отдельный фоновой цикл, не зависящий от личных напоминаний о прогнозах. После запуска Railway он сразу проверяет матчи, стартовавшие за последние 30 минут, поэтому разбор не потеряется, если релиз установлен сразу после начала матча.

Отправка защищена ключом события в БД: для одного чата, лиги и игрового слота сообщение уходит только один раз даже после перезапуска приложения.

## Миграции

Новых миграций нет. Для AI-комментариев и настроек тона должна быть уже применена миграция из v2.8.55:

```text
db/migrations/020_add_gamification_and_ai_messages.sql
```

## Переменные окружения

```env
# Включить общий AI-разбор прогнозов. По умолчанию включено.
PREGAME_ANALYSIS_ENABLED=true

# Интервал проверки старта матчей, секунд. Минимум 20.
PREGAME_ANALYSIS_CHECK_INTERVAL_SECONDS=30

# Как далеко назад при запуске проверять уже начавшиеся матчи.
# Это позволяет не пропустить разбор сразу после Railway-деплоя.
PREGAME_ANALYSIS_LOOKBACK_MINUTES=30

# Используются существующие настройки из v2.8.55.
OPENAI_API_KEY=...
OPENAI_GAMIFICATION_MODEL=gpt-5.5-mini
OPENAI_GAMIFICATION_ENABLED=true
```

Без `OPENAI_API_KEY` сообщение всё равно будет отправлено: прогнозы, таблица и форма формируются из БД, а текст AI-разбора заменяется шаблонным комментарием.

## Проверка

1. Задать `chat_id` у лиги.
2. Убедиться, что в Railway задан `OPENAI_API_KEY`.
3. После старта матча дождаться до 30 секунд: в чат придёт один разбор игрового слота.
4. Если релиз развёрнут уже после старта, разбор будет отправлен, пока матч попадает в 30-минутное окно и ещё не завершён.
