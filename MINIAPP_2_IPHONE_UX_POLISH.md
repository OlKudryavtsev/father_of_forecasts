# Mini App 2.0 iPhone and UX polish

Исправляет замечания по React Mini App 2.0.

## Что изменено

### iPhone UI

- Уменьшены и адаптированы размеры шрифтов в шапке.
- Текст `ЧМ-2026 · США · Мексика · Канада` больше не вылезает за рамки: включен `ellipsis`.
- Карточки матчей, таблицы групп и нижняя навигация адаптированы под узкий экран.
- Убраны переполнения по горизонтали.

### Шапка

- `Лига прогнозов` заменено на `Отец прогнозов`.
- Добавлен онлайн-таймер до старта турнира с обновлением раз в секунду.
- В нижнем меню `Матчи` заменено на `Матч-центр`.
- Баскетбольный мяч заменен на футбольный.

### Правила

В модальное окно `Правила` добавлен красиво оформленный текст из команды `/rules`:

- очки за точный счет;
- очки за исход;
- пример начисления;
- плей-офф;
- прогноз на итоги турнира.

### Матч-центр

- В карточки матчей добавлена кнопка `🤖 Прогноз Отца`.
- Кнопка открывает модальное окно и вызывает `/api/webapp/forecast/{match_id}`.
- Таблицы групп визуально переработаны: 3-е место больше не подсвечивается всей строкой, зоны отмечаются аккуратным индикатором слева.

### Турнир

- Исправлен прогресс `Прогнозы на турнир`: теперь Mini App дополнительно загружает `/api/webapp/tournament-prediction/me`, а не полагается только на dashboard.
- В раздел `Турнир` возвращены:
  - `Прогноз Отца прогнозов`;
  - объяснение прогноза;
  - альтернативы;
  - фирменный комментарий;
  - помощь Отца по выбору бомбардира;
  - список кандидатов на бомбардира.

## SQL

Миграции не нужны.

## Сборка

React frontend находится в:

```text
app/miniapp_frontend
```

Dockerfile собирает его автоматически и копирует результат в:

```text
app/miniapp_static
```
