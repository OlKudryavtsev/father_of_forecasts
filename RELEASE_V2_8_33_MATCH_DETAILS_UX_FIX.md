# v2.8.33 — Исправления UX деталей матча и Турнира

## Исправления

- Детали матча теперь открываются как компактный нижний лист: высота ограничена 72% окна, контент прокручивается внутри.
- Кнопка закрытия увеличена до 48×48 px; добавлены область касания и закрытие по тапу на затемнение вокруг листа.
- Убрано дублирование переключателей «Турнир» / «Бомбардиры»: остается единый переключатель в верхней части Матч-центра.
- Добавлен увеличенный безопасный нижний отступ для приложения и результатов Матч-центра, чтобы таблицы групп и длинные карточки не перекрывались нижним меню.
- Все модальные окна теперь визуально располагаются над нижней навигацией.

Не требуются.
