# Реализация `/admin_release`

Добавлена админская команда для отправки Release Notes от имени бота.

## Команда

```text
/admin_release group
текст релиза
```

Режимы:

```text
group — отправить в общий чат из GROUP_CHAT_ID
users — отправить всем зарегистрированным пользователям в личку
both — отправить и в общий чат, и пользователям
```

## Пример

```text
/admin_release group
🚀 Release Notes v0.7

Добавлено:
— /quiz_battle — серия квиза на 3/5/10 вопросов
— /table_buttons — компактная таблица кнопками
```

## Измененные файлы

```text
app/handlers/admin.py
app/bot_runtime.py
README.md
RELEASE_NOTES.md
```

## Технические детали

- Команда доступна только администраторам.
- В групповые команды она не добавлена, чтобы обычные участники ее не видели и не вызывали.
- Для отправки в личку бот проходит по таблице `users`.
- Если пользователь не открывал личный чат с ботом или заблокировал бота, отправка ему может завершиться ошибкой; такие ошибки логируются и считаются в итоговом отчете.
- Длинные сообщения режутся на Telegram-safe chunks.
