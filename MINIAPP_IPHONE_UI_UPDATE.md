# Mini App iPhone UI update

Обновление улучшает отображение Telegram Mini App на iPhone и маленьких экранах.

## Что изменено

- Переработана нижняя навигация: фиксированная bottom-tab панель с учетом safe-area iPhone.
- Улучшены карточки матчей, отступы, шрифты, радиусы и touch targets.
- Кнопки стали полноширинными на телефоне и не “схлопываются”.
- Переработан редактор счета: теперь это два понятных блока `− счет +`.
- Таблица на iPhone показывается карточками участников, а не широкой таблицей.
- Для desktop / широких экранов сохранен табличный режим.
- Добавлены iOS meta tags в `index.html`.
- Сохранен фикс: форма прогноза больше не пропадает после открытия.

## Файлы

- `app/miniapp_static/index.html`
- `app/miniapp_static/styles.css`
- `app/miniapp_static/app.js`

SQL и backend не менялись.
