# Отец прогнозов v2.8.66 — сетка плей-офф

## Что добавлено

В **Матч-центре → Турнир** появился второй компактный раздел — **Плей-офф**.

- Стадии доступны отдельными переключателями: `1/16`, `1/8`, `1/4`, `1/2`, матч за 3-е место и финал.
- Каждая карточка матча показывает дату, город, команды и итоговый счёт после завершения.
- Когда один из участников ещё неизвестен, слот не выглядит как пустой `TBD`: в нём отображается, из какого предыдущего матча приходит победитель, и перечисляются возможные команды.
- Ниже добавлена полная турнирная сетка в горизонтальном скролле. Она повторяет логику перехода победителей между стадиями; карточку матча можно открыть нажатием.

## Источник данных

Сетка строится по матчам, уже сохранённым в локальной базе и синхронизированным из API-Football. Для 1/16 → 1/8 используется опубликованная структура ЧМ-2026; для поздних раундов предусмотрен безопасный fallback по порядку матчей.

## Деплой

Миграции и новые переменные окружения не нужны. После деплоя Mini App загрузит новый блок из существующего `GET /api/webapp/tournament/overview`.
