# Match participants visibility update

## Что изменено

1. На вкладке `Матч-центр` блок `Мнение участников` теперь скрыт до начала конкретного матча.
   - До начала матча распределение П1 / X / П2 не показывается.
   - После начала матча распределение снова отображается.

2. В карточке матча добавлена кнопка `Участники`.
   - До начала матча показывает только список участников, которые уже сделали ставку.
   - Прогнозные счета до начала матча не раскрываются.
   - После начала матча показывает участников и их прогнозы.

3. Добавлен endpoint:

```http
GET /api/webapp/matches/{match_id}/predictions
```

Он возвращает прогнозы только после стартового времени матча.
