# v2.8.38 — League activity privacy fix

## Исправление

История действий лиги больше не показывает счет, который участник поставил на матч.

Теперь в ленте отображается только факт действия:

- «Сделал прогноз»;
- «Изменил прогноз».

Для контекста сохраняется только название матча, без выбранного счета.

## Технически

- новые события больше не сохраняют прогнозируемый счет в payload;
- старые события с уже сохраненным счетом также не раскрывают его через API;
- миграции БД не требуются.
