# v2.8.34 — Tournament UX refinements

## Исправления

- переключение «Матчи / Турнир / Бомбардиры» обновляет только рабочую область ниже переключателя;
- добавлен выбор нескольких групп через выпадающий список; по умолчанию выбраны группы с матчами текущего игрового дня по `America/New_York`;
- выводятся все выбранные таблицы групп;
- таблицы групп уплотнены для вертикального iPhone;
- контент вкладок «События» и «Статистика» в деталях матча отделен от панели вкладок и больше не накладывается на нее;
- в топе бомбардиров оставлен компактный флаг сборной без названия, чтобы флаг был виден и в вертикальной ориентации iPhone;
- усилен нижний прокручиваемый отступ рабочего содержимого, чтобы карточки не скрывались за нижней навигацией.

## Данные

Миграции не нужны. Backend добавляет в ответ `/api/webapp/tournament/overview` поле `default_group_codes`.
