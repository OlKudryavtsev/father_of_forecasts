# Mini App: главная, турнирный прогноз и AI-прогнозы

## Что добавлено

### 1. Главная

На главной странице появились напоминания:

- до старта ЧМ-2026 осталось N дней, если пользователь еще не сделал турнирный прогноз;
- есть матчи ближайшего игрового дня без прогноза.

В обоих блоках есть кнопки перехода к нужному действию:

- `Перейти к турнирному прогнозу`;
- `Сделать прогнозы на ближайшие матчи`.

### 2. Турнирный прогноз

Поля:

- чемпион;
- финалист;
- 3 место.

Теперь выбираются через autocomplete-список сборных. Список строится backend'ом из матчей текущего турнира (`matches`) и фильтруется по введенному тексту на клиенте.

Поле `Бомбардир` осталось свободным текстовым полем.

### 3. Прогнозы

На карточке матча и в форме прогноза добавлена кнопка:

```text
🤖 Прогноз Отца
```

Она вызывает backend endpoint:

```text
GET /api/webapp/forecast/{match_id}
```

Endpoint использует ту же AI-логику, что и команда `/forecast` в Telegram-боте.
