# Admin button, Fantasy random button and Profile collapsible sections

## Что изменено

### 1. Кнопка `Админ`

- `Админ` теперь отдельная кнопка слева от блока обратного отсчета.
- Кнопка больше не находится внутри прямоугольника обратного отсчета.
- Кнопка видна только администраторам из `ADMIN_TELEGRAM_IDS`.

### 2. Fantasy

- Кнопка `случайный состав` скрывается, если состав уже собран полностью: 15/15 игроков.
- Если состав неполный, кнопка остается доступной.

### 3. Профиль

Разделы профиля сделаны сворачиваемыми с сохранением текущего карточного стиля.

Новый порядок разделов:

1. Настройки
2. Откуда очки
3. Статистика
4. Достижения
5. Прогнозы на турнир
6. Уведомления

Каждый раздел можно раскрыть/свернуть нажатием на заголовок.
