# Mini App 2.0 round 2 polish

Исправления по замечаниям после проверки React Mini App 2.0 на iPhone.

## Что изменено

1. Нижнее меню:
   - уменьшены подписи;
   - `Матч-центр` теперь помещается;
   - emoji-иконки заменены на inline SVG.

2. Шапка:
   - убрана верхняя дублирующая строка `Отец прогнозов · ЧМ-2026`;
   - оставлена одна шапка с `Отец прогнозов`;
   - кнопка `Правила` больше не вылезает за рамки;
   - таймер обновляется онлайн раз в секунду.

3. Время старта ЧМ:
   - дефолтное значение `TOURNAMENT_STARTS_AT` обновлено с `2026-06-11T21:00:00+03:00` на `2026-06-11T22:00:00+03:00`.
   - Если в Railway уже задана переменная `TOURNAMENT_STARTS_AT`, она имеет приоритет над дефолтом в коде.

4. Фильтры матч-центра:
   - переделаны в более аккуратные квадратные кнопки;
   - убраны emoji;
   - дизайн ближе к референсу: `Все`, `Результаты`, `A группа`, `B группа` и т.д.

5. Логика фильтров:
   - `Все` показывает все матчи без таблицы группы A;
   - `Результаты` показывает только завершенные матчи и не показывает таблицу;
   - таблица группы показывается только при выборе конкретной группы.

6. Турнирные прогнозы:
   - секция добавлена на `Матч-центр`;
   - оформлена как 4 мини-квадрата:
     - Победитель +15;
     - 2-е место +10;
     - 3-е место +5;
     - Бомбардир +15.

7. Вкладка `Прогнозы`:
   - убраны переключатели `Нужен прогноз` / `Все будущие`;
   - сверху показываются счетчики:
     - `Нужен прогноз` — будущие матчи без прогноза;
     - `Можно изменить` — будущие матчи, где прогноз уже сделан;
   - ниже выводятся обе секции.

8. Карточки матчей:
   - убрано дублирование кнопок `Сделать прогноз` и `Прогноз`;
   - оставлена одна кнопка прогноза;
   - добавлена кнопка `Прогноз Отца`;
   - форма прогноза переделана под iPhone 12 Pro: команды идут строками, счетчик компактный.

9. Empty state:
   - баскетбольный мяч заменен на SVG-иконку.
