# v2.8.42 — Фильтры точных счетов и исходов в истории участника

## Что изменено

- Плашки «N точных» и «N исходов» в карточке истории прогнозов участника стали интерактивными.
- Нажатие на «точных» показывает только матчи с точно угаданным счётом.
- Нажатие на «исходов» показывает только матчи с угаданным исходом.
- Повторное нажатие на активную плашку возвращает полный список прогнозов.
- Активный фильтр мягко выделяется цветом, соответствующим типу попадания, без излишне яркого акцента.
- Добавлено понятное пустое состояние, когда для выбранного фильтра нет матчей.

## Миграции

Не требуются.
