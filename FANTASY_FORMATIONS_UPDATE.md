# Fantasy formations update

## Что изменено

В Fantasy добавлен выбор схемы стартового состава.

Поддерживаемые схемы:

- 4-3-3
- 4-4-2
- 4-2-2-2 — в интерфейсе отображается как 4-2-2-2, техническое значение `4-2-2`
- 5-4-1
- 4-5-1
- 3-5-2
- 3-4-3
- 5-3-2
- 4-1-4-1

## Логика

- Заявка по-прежнему состоит из 15 игроков:
  - 2 ВР
  - 5 ЗЩ
  - 5 ПЗ
  - 3 НП
- Пользователь выбирает схему основы.
- При смене схемы выбранные игроки переносятся в новые слоты по позициям, насколько это возможно.
- Игроки, которые не помещаются в основу новой схемы, переносятся на скамейку.
- Капитан сбрасывается, если после смены схемы он оказался не в основе.
- Backend теперь валидирует стартовый состав по выбранной схеме, а не только по 4-3-3.

## SQL

Миграции не нужны.
Поле `fantasy_teams.formation` уже есть и используется.
