# v2.6 Match start predictions and daily match summary

## Что изменено

### 1. Уведомление о старте матча

Групповое уведомление о старте матча теперь включает:

- матч;
- прогноз Отца прогнозов;
- прогнозы всех участников на матч.

Прогноз Отца берется из `father_match_predictions`; если его еще нет, он создается и фиксируется. Для первых двух матчей сохраняется логика 1:0 и 1:1.

### 2. Утренняя ежедневная рубрика

Ежедневное утреннее сообщение больше не отправляет:

- Факт дня;
- Отец прогнозов;
- Архив Отца прогнозов.

Вместо этого отправляется утренний свод за последние 24 часа:

- завершенные матчи;
- итоговый счет;
- кто угадал точный счет;
- кто угадал исход;
- кто не угадал;
- юморной комментарий Отца прогнозов.

Если матч за последние сутки был, но результат еще не внесен, он попадает в блок «Матчи без финального результата».

### 3. Админские команды ежедневной рубрики

Команды предпросмотра/ручной отправки ежедневной рубрики теперь также используют новый матчевый свод, а не факт дня.
