# v2.8.57 — Прогноз Отца в разборе и история ИИ-прогнозов

## Что изменено

- В групповом сообщении **«🔥 Прогнозы вскрыты»** для каждого матча вновь отображается строка с зафиксированным прогнозом Отца: `🤖 Отец прогнозов: X:Y`.
- Разбор Отца теперь строится как единый комментарий о картине прогнозов всей лиги: едином мнении, расколе, популярных сценариях, различиях в дуэлях и общем количестве необычных выборов. Один участник больше не становится единственной темой сообщения.
- Для OpenAI-разбора добавлен ключ вариативности игрового слота и запрет на прежнюю повторяющуюся формулировку.
- Когда OpenAI недоступен, резервный текст тоже формируется по общей картине слота и меняет подачу между слотами.
- По умолчанию для AI-комментариев используется та же модель, что настроена для прогнозов Отца (`OPENAI_FORECAST_MODEL`), если отдельная `OPENAI_GAMIFICATION_MODEL` не задана. Это уменьшает риск перехода на шаблонный fallback из-за отсутствующей модели.
- Плашка **«🤖 Отец прогнозов»** во вкладке «Рейтинг» стала кликабельной. Открывается история завершённых прогнозов Отца с итогами, прогнозами, очками, точными счетами и угаданными исходами.

## Миграции и настройки

Миграции не нужны. Новые переменные окружения не нужны.

Для вариативного AI-разбора должны быть заданы `OPENAI_API_KEY` и либо уже используемая `OPENAI_FORECAST_MODEL`, либо отдельная `OPENAI_GAMIFICATION_MODEL`. При недоступности OpenAI сообщение всё равно отправится с динамическим безопасным шаблоном.
