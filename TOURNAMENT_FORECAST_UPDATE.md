# Tournament Forecast update

Добавлена функциональность «Прогноз Отца» на турнирные итоги ЧМ-2026.

## Что добавлено

### Бот

Новая команда:

```text
/tournament_forecast
```

Показывает статичный прогноз Отца прогнозов:

- чемпион;
- финалист;
- 3 место;
- лучший бомбардир;
- объяснение;
- альтернативы;
- уверенность;
- качество данных;
- фирменный комментарий Отца.

Команда доступна и в личке, и в групповом чате.

### Mini App

На вкладке `Турнир` добавлена карточка:

```text
🤖 Прогноз Отца на турнир
```

Карточка компактно показывает прогноз, а подробности раскрываются по кнопке `Показать объяснение`.

### Выбор бомбардира

В Mini App поле `Бомбардир` заменено на выбор из списка кандидатов:

- Килиан Мбаппе;
- Харри Кейн;
- Эрлинг Холанд;
- Лионель Месси;
- Ламин Ямаль;
- Криштиану Роналду;
- Лаутаро Мартинес;
- Винисиус Жуниор;
- Усман Дембеле;
- Букайо Сака;
- и другие.

Также есть вариант `Свой вариант`, если участник хочет указать другого игрока.

### Подсказка по бомбардирам

Добавлена кнопка:

```text
Подсказка по бомбардирам
```

Она раскрывает короткое объяснение логики выбора и карточки основных кандидатов.

## Данные

Добавлен файл:

```text
data/father_tournament_forecast_wc2026.json
```

Он содержит статичный прогноз Отца, альтернативы и список кандидатов в бомбардиры.

## API Mini App

Добавлены endpoints:

```text
GET /api/webapp/tournament-forecast
GET /api/webapp/top-scorer-candidates
```

SQL не менялся.
