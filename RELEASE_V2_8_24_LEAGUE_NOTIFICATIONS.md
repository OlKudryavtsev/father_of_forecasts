# v2.8.24 — League notifications

## Что изменилось

- Добавлена настройка `chat_id` для лиги: владелец/администратор лиги может указать Telegram chat ID группы в разделе **Лиги → Управлять лигой**.
- Уведомления в `GROUP_CHAT_ID` теперь остаются только для системной лиги **«Отец прогнозов»** и ее участников.
- Если у частной лиги указан `chat_id`, уведомления по событиям этой лиги отправляются в ее чат:
  - прогноз участника лиги;
  - турнирный прогноз участника лиги;
  - старт матча с прогнозами участников этой лиги;
  - итог матча с результатами участников этой лиги;
  - ежедневная сводка по участникам этой лиги.
- Личные уведомления через бота теперь отправляются approved-пользователям по умолчанию с учетом пользовательских настроек уведомлений.
- Добавлено персональное напоминание за 1 час до матча независимо от того, сделан ли прогноз: «Ты уверен, что сделал верный прогноз? Может, поменяешь?».

## Настройки

Новая миграция:

```bash
psql "$DATABASE_URL" -f db/migrations/014_league_chat_notifications.sql
```

Новая опциональная переменная:

```bash
CONFIDENCE_REMINDER_OFFSET_MINUTES=60
```
